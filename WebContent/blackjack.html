BlackJack!

BACKGROUND

BlackJack! is a standalone Java application I developed that allows 1-3 users connected on the same server to play a modified version of the classic card game, BlackJack. It was a great chance to gain experience in implementing server sockets to allow different clients on the same network to partake in the same game. Currently, BlackJack! does not have a user-friendly GUI and is displayed in the console. I hope to make time in the future to create a GUI for BlackJack! as another project!

TECHNICAL DETAILS

To play BlackJack!, the user initially starts a server by inputting an available port on which the program will instantiate a server socket. Once created, clients can connect to the server and create games with unique room names, ranging in size from 1 - 3 players.

When the room reaches capacity, the players are prompted in turns to make bets. Each player starts with 500 coins and the game ends when one player loses all their coins. Once bets have been placed by all players, all players and the dealer are dealt 2 random cards each. Players then take turns deciding either to stay or hit, adding another random card to their hand and being prompted once again if choosing the latter.

Once each player has decided to either stay or busts and loses the turn, the automated dealer takes its turn. To implement the dealer's logic, I created an algorithm in which the dealer will hit until his hand value is greater than or equal to 17. This simplified the decision logic and made it easier for players to beat the dealer. When the dealer's turn is over, the round ends, and players are either awarded or deducted their bet amounts, depending on whether their hand beat the dealers.

CONCLUSION

Creating the BlackJack! program turned out to be much harder than I had originally expected it to be. It was an interesting experience working with server sockets and adding network functionality so that different clients could communicate with each other when connected on the same server. Working on this project with networked programming was a greatly educational and informative encounter and I look forward to working with networked programming further in the future!
